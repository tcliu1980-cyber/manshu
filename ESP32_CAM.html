ESP32 and ESP32-CAM Development Guide

1. ESP32-CAM 硬體架構

ESP32-CAM 是一款整合了 ESP32 晶片與 OV2640 攝影機模組的微型開發板。由於其體積限制，並未內建 USB 燒錄晶片。

2. 程式燒錄接線步驟

請使用 USB 轉 TTL (USB-to-TTL) 線進行連接，並依照下表完成電路跳線：

USB-to-TTL 腳位 ESP32-CAM 腳位 功能說明
5V 5V 電源輸入 (Power)
GND GND 共地 (Ground)
TXD U0R 資料接收 (Receive)
RXD U0T 資料傳送 (Transmit)
GND GPIO 0 燒錄模式跳線 (Flash Mode Jumper)

上傳程式作業流程 (Upload Workflow)

  1. 進入燒錄模式：務必將 GPIO 0 與 GND 用跳線連接（短路）。
  2. 通電：將 USB-to-TTL 插入電腦。
  3. 編譯與上傳：在 Arduino IDE 按下上傳。若出現「Connecting...」，請按下開發板背面的 Reset 鍵。
  4. 回復執行模式：上傳完成後，移除 GPIO 0 的跳線，並再次按下 Reset 鍵。

3. 基礎規格概覽 (Technical Specs)

  • Wi-Fi: 802.11 b/g/n/e/i
  • Bluetooth: Bluetooth 4.2 BR/EDR and BLE standards
  • Memory: Internal 520KB SRAM + External 4MB PSRAM
  • Storage: MicroSD card slot supported
